# LM Web App

এই ওয়েব অ্যাপটি ব্যবহারকারীদের **ছবি এবং ভিডিও আপলোড ও প্রসেসিং** করার সুযোগ দেয়।  
সহজে ব্যবহারযোগ্য ইন্টারফেস এবং ডাউনলোড ফিচার সহ এটি দ্রুত এবং নিরাপদ।  

---

## Features
- ছবি আপলোড করে প্রক্রিয়াজাতকরণ (AI প্রক্রিয়া)
- ভিডিও আপলোড ও প্রক্রিয়াজাতকরণ (iPhone 17 PMIX স্টাইল)
- ডাউনলোড বোতাম থেকে প্রক্রিয়াজাত ফাইল ডাউনলোড করা যাবে
- মোবাইল এবং ডেস্কটপে সম্পূর্ণ রেস্পন্সিভ

---
